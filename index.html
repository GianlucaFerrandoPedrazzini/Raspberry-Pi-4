Pygame

Biblioteca de código abierto para crear videojuegos y aplicaciones multimedia con Python

¿Qué es Pygame?

Pygame es una biblioteca gratuita y de código abierto que permite programar aplicaciones multimedia y videojuegos utilizando el lenguaje Python. Es utilizada tanto para aprender programación como para desarrollar juegos completos.

Trabajadores

El proyecto es mantenido por un grupo de voluntarios humanos, entre los que también se mencionan de forma humorística gatos, koalas, perros, 3.14 gnomos y 42 robots que colaboran con el desarrollo.
